# Movie Recommendation System

## Project Overview

The purpose of this project is to develop a web-based movie recommendation system that helps users search for and discover movies more easily. The system allows users to register and log in, manage their personal information, search for movies by tags, save favourite movies, and receive automatic movie recommendations based on their preferred tags.

The website will also provide an administrator backend that allows administrators to add, delete, and update movie information. This helps keep the movie database accurate, organised, and useful for users.

## Objectives

* To develop a movie recommendation system that allows users to search for movies online.
* To help users find movie information quickly based on tags and preferences.
* To provide personalised movie recommendations based on users’ favourite tags.
* To allow users to save their favourite movies for future viewing.
* To provide administrators with tools to manage movie information efficiently.

## Main Features

* User registration and login
* User personal information editing
* Movie search by tags
* Movie information display
* Favourite movie collection
* Automatic movie recommendation based on user preference tags
* Administrator backend
* Add movie information
* Delete movie information
* Update movie information

## Technology Stack

* Front-end: HTML, CSS, JavaScript
* Back-end: PHP
* Database: MySQL
* Version Control: GitHub
